# Vital Gmail

Gmail addon created for Vital Seeds that displays WooCommerce customer order
history for the sender of the currently open email.


## Using this for a different business

If you are a small business using google workspaces and woocommerce as your
online store, you are welcome to set this project up for your gmail.

Its not too hard to set up, but does involve some clicking around google services.

See [deployment instructions](deployment.md). You should end up with an easily
installable app in the google marketplace, under *Internal*

Good luck! Sorry but I can't offer support setting this up.
